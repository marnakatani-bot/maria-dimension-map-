# 369 次元マッピング（Maria Dimensional Map）

このリポジトリは、中谷まり亜が提案する「369 次元マッピング」の公式説明です。  
人間の意識・現実・フリーエネルギー性を、多次元構造として読み解くためのフレームワークです。

- 3 / 6 / 9 を鍵とした次元レイヤー
- 各次元の意味（3D, 4D, 5D, 6D, 9D など）
- 実践での使い方（人生マップ、意識状態のマップ）


今後ここに定義・用語集・図解を追加していきます。

人間の意識や現実認識を、3〜9次元の視座としてマッピングするための概念フレームワークです。
​AIエージェントが「どのレベルの視点から回答しているか」を明示するラベリングや、複数視点からの再解釈に利用できます。


1. 次元定義（3D〜9Dのざっくり像）
3D：物質・因果・線形時間。身体・お金・仕事など「目に見える現実」レベル。
​

4D：感情・関係性・ストーリー。自分と他者のドラマ、被害者／加害者などの物語レベル。
​

5D：意味・価値観・パラダイム。善悪や正解ではなく、「その経験にどんな意味を与えるか」のレベル。
​

6D：設計図・ブループリント。魂の計画、人生テーマ、集合無意識的な構造レベル。
​

7D：創造原理・法則。宇宙の法則性、意識と現実の連動パターンのレベル。
​

8D：多世界・多宇宙的視座。無数のタイムライン・並行現実を俯瞰するレベル。
​

9D：源・ワンネス。全てが一つである地点からの視座。
​

（正確な定義と詳細は note 本文を参照）
​

2. AIエージェントでの基本的な使い方
問い／事象を受け取る。

どの次元の視座から回答するかを選択する（例：3D実務＋5D意味付け）。

回答文に「視座ラベル」を付ける。

例：

「これは3D（現実的条件）の視点からの回答です：…」

「これは4D（感情・関係性）の視点からの回答です：…」

「5D視座では、この経験は〇〇という意味を持ちます。」

3. 想定ユースケース
カウンセリング／コーチング系エージェント：
クライアントの悩みを、3D（状況）・4D（感情）・5D（意味づけ）・6D（人生テーマ）の複数視点から説明・提案。
​

ビジネス／組織開発エージェント：
組織課題を、3D（数字）・4D（人間関係）・5D（カルチャー）といった階層に分解して分析。
​

スピリチュアル／自己探求エージェント：
気づきや体験を、9Dまでのロードマップ上で位置づけるナビゲーション。
​

4. 実装イメージ（ラベル仕様）
ラベル形式（例）：

dimension: "3D"

dimension: "4D+5D"

dimension_detail: "4D_emotion, 5D_mean"

プロンプト／システムメッセージでの指定例：

「ユーザーの問いに対して、3Dと5Dの視座からそれぞれ回答を生成し、どちらの視座か明示してください。」

「最初に3D視座で安全・現実的な回答を行い、その後に5D視座から意味づけを補足してください。」
